MyFlix API

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description of Endpoint Return Endpoint Method Format of Request Data Format of Response Data
Return a list of all movies /movies GET None A JSON object holding data about all movies
Return data about a single movie by title /movies/:Title GET None A JSON object holding data about the requested movie ex: { "_id": ObjectId(12345678900) "Title": "The Shawshank Redemption", "Description": "Description of movie", "Genre": { "Name": "Thriller", "Description": "Description of Genre" } "Director": { "Name": "Director Name", "Bio": "Bio of Director", "Birthyear": "Birthyear of Director" } "ImageUrl": "movietitle.png", "Featured": true }
Return data about the genre by name /genre/:Name GET None A JSON object holding data about the genre ex: { "Description of Genre" }
Return data about a director by name /movies/director/:name GET None A JSON object holding data about the director ex: { "_id": ObjectId(12345678900) "Name": "Frank Darabont", "Bio": "Bio of Director", "Birthyear": "Birthyear of director", "Deathyear": "Deathyear of director" }
Create new user /users POST A JSON object holding data from the user to be added ex: { "Name": "Jane Doe", "Email": "janedoe@mail.com", "Password": "password123", "Birthdate": { "$date": "1998-08-13T00:00.000Z" }, "FavMovies": [ "The Shawshank Redemption" ] } A JSON object holding data from the user that has been added with uuid ex: { "_id": ObjectId(12345678900) "Name": "Jane Doe", "Email": "janedoe@mail.com", "Password": "password123", "Birthdate": { "$date": "1998-08-13T00:00.000Z" } "FavMovies": [ "The Shawshank Redemption" ] }
Update user /users/:Name PUT A JSON object holding data from the user to be updated ex: { "_id": ObjectId(12345678900) "Name": "John Doe", "Password": "password123", "Email": "janedoe@mail.com", "Birthdate": "1998-08-13T00:00.000Z", "FavMovies": ["The Shawshank Redemption"] } A JSON holding data from the user that has been update ex: { "_id": ObjectId(12345678900) "Name": "John Doe", "Password": "password123", "Email": "janedoe@mail.com", "Birthdate": "1998-08-13T00:00.000Z", "FavMovies": ["The Shawshank Redemption"] }
Update users favMovies /users/:Name/movies/:MovieID POST None A JSON object returning if the users favMovies have been updated ex. { "_id": ObjectId(12345678900) "Name": "John Doe", "Password": "password123", "Email": "janedoe@mail.com", "Birthdate": "1998-08-13T00:00.000Z", "FavMovies": ["The Shawshank Redemption"] }
Delete a favMovies /users/:id/:title DELETE None A message returning if the users favMovies have been updated
Delete a user /users/:id DELETE None A message returning if the user was deleted

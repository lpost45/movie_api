MyFlix API

This web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description of Endpoint Return Endpoint Method Format of Request Data Format of Response Data
Return a list of all movies /movies GET None A JSON object holding data about all movies
Return data about a single movie by title /movies/:title GET None A JSON object holding data about the requested movie ex: { "title": "The Shawshank Redemption", "director": "Frank Darabont", "genre": "Thriller" }
Return data about the genre by title /movies/:title/genre GET None A JSON object holding data about the genre of the specific movie ex: { "Thriller" }
Return data about a director by name /movies/director/:name GET None A JSON object holding data about the director ex: { "name": "Frank Darabont", "age": "65", "birthplace": "France" }
Create new user /users POST A JSON object holding data from the user to be added ex: { "username": "lpost45", "email": "lpost45@live.com" } A JSON object holding data from the user that has been added with uuid ex: { "username": "lpost45", "email": "lpost45@live.com", "id": "b8d9e773-2b36-4967-83a5-af57301053ab" }
Update username /users/:id PUT A JSON object holding data from the user to be updated ex: { "username": "lpost95" } A JSON holding data from the user that has been update ex: { "username": "lpost95", "emial": "lpost45@live.com" }
Update users favMovies /users/:id/:title POST A JSON object holding data from the user about favMovies to be updated A message returning if the users favMovies have been updated
Delete a favMovies /users/:id/:title DELETE None A message returning if the users favMovies have been updated
Delete a user /users/:id DELETE None A message returning if the user was deleted
